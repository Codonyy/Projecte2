
# README — Projectes EverPia

## Descripció del projecte

Benvinguts a **EverPia**, una consultora IT especialitzada en solucions tecnològiques per empreses.

Aquest repositori conté totes les activitats, proves de concepte, documentació tècnica i productes finals desenvolupats durant el projecte intermodular. L’objectiu principal és treballar com una consultora real, aplicant metodologies professionals, documentació tècnica i treball col·laboratiu.

Els projectes inclouen:

* Infraestructura de sistemes
* Active Directory
* Seguretat informàtica
* Servidors Linux i Windows
* Hosting i desplegament web
* Cloud computing
* Auditoria i monitorització
* Gestió documental
* Pressupostos i planificació

# Objectius del projecte

Els principals objectius d’aquest projecte són:

* Aplicar metodologies professionals de treball
* Desenvolupar infraestructures IT reals
* Millorar la seguretat i disponibilitat dels serveis
* Documentar processos tècnics amb Markdown
* Utilitzar eines de treball col·laboratiu
* Presentar propostes tècniques i econòmiques professionals
